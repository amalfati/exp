Introduction:

2048 is a popular single-player game where we combine numbered tiles in order to gain a higher numbered tile.will play by combining the tiles with the same number to create larger numbered tiles. When there are no more moves left, and the board is full, the game is over.

2048 has become so popular that many developers have implemented it in different programming languages to test their programming skills and in this report we will do it using C++ in QtCreator.

Goals:

Our goals is to be able to :

  • Program Do 4-by-4 square grid partially filled with tiles labeled with powers of 2.
  • Move left, right, up, or down, and the result should change.
  • Do a game able to calculate the number of scores.
  • !The basic goal of this game is to move numbered tiles across a grid in order to merge them into a tile with the number 2048; however, you can keep playing by making greater numbers. Each time you make a move, a new tile with the number 2 falls.
